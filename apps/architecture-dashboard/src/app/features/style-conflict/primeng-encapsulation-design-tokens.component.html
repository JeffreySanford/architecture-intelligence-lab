Encapsulation.None → Token-Based Recovery

This tab demonstrates how component styles behave globally when Angular `ViewEncapsulation.None` is enabled, then shows how localized design tokens can restore predictable PrimeNG styling inside a scoped boundary.

Token-Based Recovery

This pane uses local CSS custom properties and PrimeNG variable mapping to recover consistent styling without broad page-level overrides.

Token values are defined locally for this tab and mapped into PrimeNG CSS variables inside the demo wrapper.

Page preview

Example page content to prove the same Normal PrimeNG controls are rendered inside the token recovery tab.
